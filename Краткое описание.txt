1. Установка:
   - Убедитесь, что у вас установлен Python 3.x на вашем компьютере.
   - Скачайте или скопируйте все файлы проекта.
   - Установите необходимые зависимости, в данном случае никакие дополнительные зависимости не требуются.

2. Настройка базы данных:
   - Убедитесь, что у вас установлен SQLite.
   - Создайте базу данных SQLite с помощью следующей команды: sqlite3 notes.db.
   - В этой базе данных будут храниться заметки.

3. Запуск программы:
   - Откройте терминал или командную строку.
   - Перейдите в каталог с файлами проекта.
   - Запустите файл main.py, введите команду python3 main.py в терминале.
   - Теперь вы можете использовать консольное приложение для управления заметками.

4. Основное меню:
   - При запуске программы отобразится главное меню с пунктами действий, которые вы можете выбрать.
   - Введите номер действия, соответствующий вашему выбору.
   - Далее, следуйте инструкциям, чтобы выполнить нужные операции, такие как добавление заметки, просмотр списка заметок, поиск по ключевым словам и удаление заметки.

5. Выход из программы:
   - Чтобы выйти из программы, выберите пункт "Выйти из приложения".
   - Программа будет завершена, а соединение с базой данных будет закрыто.

6. Дополнительная настройка:
   - Вы можете изменить имя и расположение базы данных SQLite, изменив переменную db_file в файле main.py.
   - Обратите внимание, что если вы переместите базу данных, вам также придется изменить путь в других файлах, использующих базу данных.
